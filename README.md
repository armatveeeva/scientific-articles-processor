# Обработка научных статей по материаловедению

Скрипты для автоматизированного извлечения химических элементов, ключевых слов и структурирования метаданных из массива научных публикаций.

Основные задачи:
- Извлечение химических сущностей с помощью библиотеки **ChemDataExtractor**
- Генерация ключевых слов на основе полного текста статьи с использованием **YAKE**
- Группировка статей по заданным категориям (металлы, материалы и др.)
- Формирование структурированных JSON-файлов для последующего анализа

## Требования к данным

Для работы пайплайна необходимы следующие файлы в корне проекта:
- `materials.json` — результат обработки каталога материалов
- `sputnik.json` — результат обработки базы Sputnik
- `words.json` — файл с ключевыми словами для фильтрации (структура см. в `config/keywords.json.example`)

> Реальные данные не включаются в репозиторий (см. `.gitignore`). Проект носит учебный характер.
